## Usage

To use ${package}, add a dependency on ${package} and use `find_package` in 
your CMakeLists.txt file:
```
find_package(${package})
include_directories($${${package}_INCLUDE_DIRS})
target_link_libraries(my_lib $${${package}_LIBRARIES})
```
