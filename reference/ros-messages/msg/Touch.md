## Message Definition
```
bool[] electrodes
```

## Arguments
#### `electrodes`
An array of booleans that represent the eight touch sensors in this order:  
FRONT_LEFT, LEFT, REAR_LEFT, CENTER, FRONT, REAR_RIGHT, RIGHT, FRONT_RIGHT  

When set to true it means the sensor is being touched
