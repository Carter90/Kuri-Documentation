## Message Definition
```
uint8 red
uint8 green
uint8 blue
```

## Arguments
For each of the arguments the value is the intensity of the LED where zero is off and 255 is brightest.

#### `red`
Intensity of the red LED

#### `green`
Intensity of the green LED

#### `blue`
Intensity of the blue LED
